# OnlineShop
Backend приложение для интернет-магазина

Функции: 

•	JWT-аутентификация(keycloack)

•	Корзина клиентов

•	Каталог

•	Управление заказами

•	Оплата товаров (Backend)

•	REST API

•	Docker Compose

•	Docker

•	Перевод долларов в рубли

•	Загрузка изображений


Стеки технологий: 

•	Java 17

•	Spring boot 2.7.5

•	Spring security

•	Oauth 2

•	Keycloak

•	JWT Authentication

•	Spring Data Jpa

•	Hibernate

•	PostgreSQL

•	Kafka

•	Spring Cloud

•	API Gateway

•	Netflix Eureka

•	Feign

•	WebFlux

•	WebMVC

•	Swagger

•	Apache Kafaka

•	Spring Cache

•	spring-boot-starter-mail

•	Stripe
